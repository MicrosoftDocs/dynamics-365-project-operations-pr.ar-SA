<span data-ttu-id="e11e0-109">في صفحة **المشاريع**، على علامة التبويب **تعقب**، يمكنك تحديد **تعقب‏‎** > **التكلفة** لفتح طريقة عرض **تعقب التكلفة** ورؤية تقدم إنفاق العمالة على كل مهمة في خطة المشروع.</span><span class="sxs-lookup"><span data-stu-id="e11e0-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="e11e0-110">تقارن طريقة العرض هذه أيضًا تكلفة العمالة الفعلية التي تم إنفاقها على مهمة بتكلفة العمالة المخطط لها على المهمة.</span><span class="sxs-lookup"><span data-stu-id="e11e0-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="e11e0-111">يستخدم Project Operations الصيغ التالية لحساب مقاييس تكلفة العمالة:</span><span class="sxs-lookup"><span data-stu-id="e11e0-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>
